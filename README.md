# 📝 NotePro

**NotePro** — це легкий та функціональний веб-застосунок для керування персональними завданнями з підтримкою форматування Markdown, автозбереженням та адаптивним інтерфейсом.

Репозиторій створено в межах навчальної практики з дисципліни **"Технології розробки програмного забезпечення"** (II семестр 3 курсу).
---

## 🚀 Про проект

Застосунок розроблений для користувачів, яким потрібен швидкий доступ до своїх завдань без необхідності встановлення додаткового ПЗ.

### Ключові можливості:
* ✅ **Task Management:** Створення, редагування та видалення завдань (CRUD).
* 📝 **Markdown Support:** Повноцінне форматування описів завдань.
* 💾 **Smart Storage:** Автозбереження змін у LocalStorage.
* 🌓 **Themes:** Світла та темна теми для комфортної роботи.
* 📱 **Responsive Design:** Адаптація під мобільні пристрої та ПК.
* 📥 **Export/Import:** Збереження та завантаження списків у форматі `.md`.

## 🏗 Структура проекту та документація

Процес розробки базується на методології **MSF (Microsoft Solutions Framework)** і розділений на 5 ключових етапів:

1.  **[Envisioning](./docs/1.Envisioning/)** (Аналіз) — Концепція, структура команди та аналіз ризиків.
2.  **[Planning](./docs/2.Planning/)** (Планування) — Функціональна специфікація, архітектура та план проекту.
3.  **[Developing](./docs/3.Developing/)** (Розробка) — Програмний код (папка `src`) та сценарії тестів.
4.  **[Stabilizing](./docs/4.Stabilizing/)** (Стабілізація) — Тестування, баг-репорти та пілотне впровадження.
5.  **[Deploying](./docs/5.Deploying/)** (Впровадження) — Фінальний звіт, постпроектний аналіз та презентація.
