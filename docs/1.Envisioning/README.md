# Етап 1: Концептуальне проектування (Envisioning)

Цей розділ містить документацію початкового етапу аналізу проекту **NotePro**. Метою етапу було визначення бізнес-цілей, формування команди та оцінка ключових ризиків.

## 🛠 Виконані роботи на 1-му тижні

1.  **Формування команди:** Визначено назву команди (**Programists**) та обрано ролі згідно з методологією MSF.
2.  **Визначення бачення:** Сформульовано основну ідею застосунку — зручний менеджер завдань з підтримкою Markdown.
3.  **Аналіз ризиків:** Проведено мозковий штурм, ідентифіковано технічні та часові ризики.
4.  **Проектування діаграм:** Створено діаграму прецедентів (Use Case) для опису взаємодії користувача з системою.
