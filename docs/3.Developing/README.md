## 🛠 Виконані роботи на 3-му тижні

1.  **Написання коду (Development):** Основна логіка застосунку реалізована та розміщена у папці [/src/](../../src/). Реалізовано CRUD операції для завдань та інтегровано Markdown-парсер.
2.  **Актуалізація документації:** Функціональна специфікація була доопрацьована, щоб відповідати реальному стану коду та архітектурним рішенням, які виникли в процесі розробки.
3.  **Створення тест-кейсів:** Тестувальники розробили покрокові сценарії для перевірки кожної функції (створення нотатки, перемикання тем, експорт у .md).
4.  **Фіналізація UI:** Створено остаточний вигляд інтерфейсу, що базується на прототипах другого тижня, але з повною кольоровою схемою та адаптивною версткою.
5.  **Підготовка Help-файлів:** Створено довідку для кінцевого користувача, що пояснює роботу з основними функціями NotePro.
