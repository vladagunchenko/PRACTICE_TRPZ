## 🛠 Виконані роботи на 2-му тижні

1.  **Архітектурне проектування:** Побудовано діаграму класів для визначення логіки роботи з завданнями та Markdown-контентом.
2.  **Деталізація вимог:** Створено функціональну специфікацію, що об'єднує бізнес-цілі з технічною реалізацією.
3.  **Планування QA:** Визначено методи та інструменти тестування для забезпечення стабільності застосунку.
4.  **Управління ризиками:** Розроблено детальний план моніторингу та нівелювання ризиків, визначених на етапі аналізу.
5.  **Розробка графіку:** Сформовано зведений план проекту з чіткими дедлайнами для кожного етапу.
